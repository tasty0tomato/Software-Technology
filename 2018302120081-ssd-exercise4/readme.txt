1. 前提
    使用Java环境，openjdk version "1.8.0_275"
    使用tomcat版本，7.0.106
    使用IDE版本，IntelliJ IDEA 2020.2.4 (Community Edition)
    使用浏览器，Chromium Version 87.0.4280.66 (Official Build) (64-bit)
2. 源文件作用
    Catfish.java catfish的方法以及变量，用于描述catfish的动作。
    HtmlAnchor.java 创建了跳转标签。
    HtmlImage.java 创建了图片标签。
    HtmlPage.java 创建了p标签。
    HtmlTable.java 创建了table标签。
    LivingBeing.java 抽象的生物类，用于继承。
    Simulation.java 应该是初始界面的构成。
    SimulationServlet.java 项目的servlet文件，用于构建catfish界面。
    SimulationView.java 应该是主界面的逻辑与变化。
    MySimulation.java 应该是主界面的构造。
    web.xml 用于配置servlet。
    initialWorld.html 创建界面。
    MySimulation.html 展示界面。
3. 修改了的源文件
    Catfish.java 补全了方法。
    SimulationView.java 补全了图片路径。
